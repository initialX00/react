ㅇ npm install react-router-dom
 (설치후 pacage.json에     "react-router-dom": "^7.1.3")

ㅇ react router dom v6 튜토리얼로 공부 (v7를 사용하지만 v6과 공유한다)
(component나 switch 키워드는 5버전이다)
https://velog.io/@velopert/react-router-v6-tutorial


ㅇ npm install @emotion/react
   emotion에 의해 문자열이 css로 전환된다.
   css임에도 조건문 등 작성이 가능해진다.

ㅇ /**@jsxImportSource @emotion/react */로 css속성이 생긴다.
   확장프로그램 vscode-styled-components 설치하여 css코드 자동완성 추가.

ㅇ단축키 임의설정
  file- preperences - configure snippets - javascriptreact에서 다음 코드 추가
	"Emotion": {
		"print to console": {
			"prefix": "ej",
			"body" : [
				"/**@JsxImoprtSource @emotion/react */",
				"import * as  s from  './style;'"
			],
			"description": "jsx @emotion/react import"
		}
	}
