*프로젝트 생성 (최신버전 설치, 19버전)

npm 오류가 뜸
(리액트 18에서 19로 넘어오면서 10.9.0버전 미지원하기에 18버전으로 다운그레이드, 19버전은 아직 문제가 많음)

*18버전으로 다운그레이드
폴더열기로 react-study로 이동 후 입력

터미널에 npm install 입력하면 package.json에 등록된 라이브러리가 자동 설치된다.
