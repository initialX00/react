*프로젝트 생성 (최신버전 설치, 19버전)
npx create-react-app react-study

*18버전으로 다운그레이드
프로젝트 폴더로 이동 후

npm install react-dom@18
npm install @testing-library/jest-dom@5.17.0

npm install @testing-library/user-event@13.5.0
npm install web-vitals

*프로젝트 실행
프로젝트 폴더로 이동 후
npm start
